Aufgabe 1:
Das Spiel lässt rechts Rechtecke spawnen, die eine Melodie darstellen, und nach links durch das Bild fliegen. Links wird durch ein rosanes Rechteck gekennzeichnet, welche Frequenz gerade durch das Mikrofon aufgenommen wird. Singt/spielt man den richtigen Ton, wird der score hochgezählt und das fliegende Rechteck wird grün.

Aufgabe 2:
Durch einen upward oder downward chirp wird der Key nach oben oder unten gedrückt. Im Terminal wird geprintet, sobald ein chirp detected wurde.
